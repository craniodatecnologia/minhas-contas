Política de Privacidade

Última atualização: 22 de dezembro de 2024

Introdução

Bem-vindo à Política de Privacidade do aplicativo Minhas Contas Premium, desenvolvido por Crânio da Tecnologia. Sua privacidade é importante para nós. Este documento descreve como coletamos, usamos e protegemos suas informações.

Dados Coletados

O aplicativo Minhas Contas Premium não coleta dados pessoais diretamente. Todos os dados inseridos pelos usuários (como despesas, receitas e configurações) permanecem localmente no dispositivo e não são enviados para servidores externos.

Uso dos Dados

Os dados inseridos no aplicativo são usados exclusivamente para fornecer as funcionalidades oferecidas, como gerenciamento de finanças pessoais. O aplicativo não compartilha, vende ou transfere essas informações para terceiros.

Compartilhamento de Dados

O Minhas Contas Premium não compartilha dados com terceiros. Caso futuramente sejam integrados serviços externos (exemplo: backup em nuvem), informaremos previamente os usuários.

Segurança dos Dados

O aplicativo adota medidas para proteger os dados armazenados localmente no dispositivo. Recomendamos que os usuários mantenham seus dispositivos protegidos com senhas e atualizações regulares.

Retenção e Exclusão dos Dados

Os dados inseridos permanecem armazenados no dispositivo do usuário até que sejam manualmente excluídos pelo próprio usuário ou até que o aplicativo seja desinstalado.

Alterações nesta Política

Podemos atualizar esta política de privacidade periodicamente. Qualquer alteração será comunicada por meio desta página.
