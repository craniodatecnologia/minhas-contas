Política de Privacidade

Última atualização: 19/12/2024

1. Introdução

Bem-vindo aos aplicativos Minhas Contas e Minhas Contas Premium, desenvolvidos pela Crânio da Tecnologia. Nossa missão é proteger sua privacidade e garantir que seus dados estejam seguros. Esta política descreve como tratamos os dados coletados e usados pelos aplicativos.

2. Coleta e Uso de Dados

Os aplicativos Minhas Contas e Minhas Contas Premium não coletam, acessam ou compartilham dados pessoais ou sensíveis, como nome, e-mail ou localização. Todos os dados financeiros inseridos pelo usuário são armazenados exclusivamente no dispositivo.

3. Permissões

Os aplicativos solicitam a permissão de armazenamento apenas para possibilitar o backup local dos dados financeiros, como despesas e receitas. Essa permissão garante que seus dados possam ser restaurados em caso de necessidade.

4. Armazenamento e Segurança de Dados

  • Os dados são armazenados localmente no dispositivo do usuário.
  • Não há transferência de dados para servidores externos ou terceiros.
  • Medidas de segurança são adotadas para proteger os dados contra acessos não autorizados.

Recomendamos que o usuário proteja seu dispositivo com senhas e mantenha o sistema operacional atualizado para garantir maior segurança.

5. Retenção e Exclusão de Dados

Os dados inseridos nos aplicativos permanecem armazenados no dispositivo enquanto o aplicativo estiver instalado. Ao desinstalar o aplicativo, todos os dados são removidos automaticamente do dispositivo.

6. Alterações nesta Política de Privacidade

Esta política pode ser alterada periodicamente. Quaisquer mudanças relevantes serão comunicadas por meio de atualizações no aplicativo ou na página do Google Play. Recomendamos que você reveja esta política regularmente.

7. Contato

Se tiver dúvidas ou preocupações sobre esta política de privacidade, entre em contato com a Crânio da Tecnologia pelo e-mail: craniodatecnologia@gmail.com
